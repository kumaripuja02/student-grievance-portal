# 🎓 Student Grievance Portal

A full-stack web application that allows students to submit and track grievances, while administrators efficiently manage, assign, and resolve them through a centralized dashboard.
---

## 🚀 Features

### 👨‍🎓 Student Module

* Register and login securely with JWT authentication
* Submit grievances with category, priority, and detailed description
* Track real-time grievance status updates
* View department assignments and admin remarks
* Personalized dashboard for grievance history

### 👨‍💼 Admin Module

* Secure admin authentication
* View and manage all submitted grievances
* Filter grievances by category, priority, and status
* Update grievance lifecycle (Pending → In Progress → Resolved → Rejected)
* Assign grievances to departments
* Add remarks and manage complaint records
* Analytics dashboard with grievance statistics

---

## 🛠️ Tech Stack

| Layer             | Technology                          |
| ----------------- | ----------------------------------- |
| Frontend          | HTML, CSS, JavaScript               |
| Backend           | Node.js, Express.js                 |
| Database          | MongoDB, Mongoose                   |
| Authentication    | JWT (JSON Web Tokens)               |
| Password Security | bcryptjs                            |
| Deployment        | Render (Backend), Vercel (Frontend) |
